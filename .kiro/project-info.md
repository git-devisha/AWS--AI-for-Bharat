# Split It - Bill Splitter Project

## Overview
A single-purpose website that elegantly solves bill splitting with friends.

## Tech Stack
- Pure HTML, CSS, and JavaScript (no frameworks)
- Responsive design with mobile-first approach
- Modern gradient UI with smooth interactions

## Features
- Bill amount input with currency formatting
- Quick tip selection (15%, 18%, 20%) or custom percentage
- Easy people counter with increment/decrement buttons
- Real-time calculations showing:
  - Tip amount
  - Total bill
  - Amount per person

## File Structure
- `index.html` - Main HTML structure
- `style.css` - All styling and responsive design
- `script.js` - Bill calculation logic and interactivity

## Design Principles
- Minimal and clean interface
- No external dependencies
- Instant calculations (no submit button needed)
- Mobile responsive
- Accessible color contrast and font sizes
