# Project Standards

This directory contains steering files that provide context and standards for this workspace.

## Current Project
- **Type**: Data visualization and automation scripts
- **Focus**: Creative data mashups and practical automation tools
- **Tech Stack**: Python, Streamlit, Plotly, pandas

## Coding Standards
- Use clear, descriptive variable names
- Include comprehensive error handling
- Add helpful comments and docstrings
- Follow PEP 8 for Python code
- Create modular, reusable functions

## Project Structure
- Keep scripts self-contained when possible
- Include requirements.txt for dependencies
- Provide both simple and advanced versions
- Add comprehensive README files
